# Pak Plastic Industry LLC

A modern, responsive website for a UAE-based plastic manufacturing company, built with HTML, CSS, and JavaScript.

## Overview

This website showcases Pak Plastic Industry's products, services, and commitment to quality manufacturing and sustainability. The site features a clean, professional design with smooth animations and full mobile responsiveness.

## Features

### Pages
1. **Home** - Hero section, why choose us, product categories, about preview, and CTA
2. **About** - Company overview, mission & vision, core values, and capabilities
3. **Products** - Detailed information about 6 product categories:
   - Plastic Pallets
   - Waste Bins
   - Bottle Caps & Closures
   - Household Items
   - Storage Containers
   - Safety Barriers
4. **Technology** - Manufacturing process, equipment, and innovation
5. **Sustainability** - Environmental initiatives, circular economy, and future goals
6. **Contact** - Contact form, company information, and location

### Key Features
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Interactive Elements** - Hover effects, scroll animations, and dynamic content
- **Mobile Navigation** - Hamburger menu for mobile devices
- **Contact Form** - Functional contact form with validation
- **Smooth Scrolling** - Enhanced user experience with smooth page transitions
- **Scroll-to-Top Button** - Easy navigation back to top of page
- **Counter Animations** - Animated statistics on scroll
- **SEO Friendly** - Semantic HTML structure

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome 6.4.0** - Icons
- **Google Fonts** - Typography (Segoe UI fallback)

## Installation & Usage

1. **Clone or Download** the project files
2. **Open** `index.html` in a web browser
3. **Navigate** through the site using the navigation menu

### Local Development
Simply open any HTML file in your preferred web browser. No build process or server required.

### Deployment
Upload all files to your web hosting server, maintaining the folder structure.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... */
}
```

### Content
Update text content directly in the HTML files.

### Images
Replace images in the `images/` folder with your own (maintain same filenames or update references in HTML).

### Contact Information
Update contact details in the footer and contact page:
- Address
- Phone number
- Email address
- Social media links
